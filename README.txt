BJJ Tracker PWA

Files:
- index.html
- manifest.webmanifest
- service-worker.js
- icon-192.png
- icon-512.png

Quick deploy:
1. Upload all files to any static HTTPS host.
2. Open index.html from that host.
3. On Android use Install App / Add to Home screen.
4. On iPhone open in Safari and use Share > Add to Home Screen.

Notes:
- Data is stored locally in the browser on this device.
- Use the app's export/import tools to back up or move data.
